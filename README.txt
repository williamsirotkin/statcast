This is an application modeled after statcast where mlb player data can be visualized by different metrics and their stats. 

To run the application from the home directory:

1) Install python version 3 - https://www.python.org/downloads/

2) Cd to backend directory (cd ./backend)

3) Run python3 backend.py

4) Open new terminal to home directory

5) cd to ./frontend/my-app

6) Install node.js - https://nodejs.org/en/download/package-manager

7) run npm i then npm start

8) Disable CORS policy in browser
    a) You can run Safari and disable CORS by clicking on preferences -> Advanced -> Disable CORS or do it from develop menu

    b) OR follow this article - https://medium.com/@beligh.hamdi/run-chrome-browser-without-cors-872747142c61

9) Go to localhost:3000

10) search player in the player list and see their stats!
